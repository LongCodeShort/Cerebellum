### find：查找指定目录下的文件

- 基本语法：`find 目录 [选项] [输出方式]` 
- 目录：指定要查询的目录，目录为空时查询当前目录
- 选项：
  1. 选项为空时，输出当前目录下的所有目录和文件
  2. `-name`：按照文件名查找
  3. `-perm`：按照安装权限进行查找
  4. `-prune`：不在当前指定的目录下寻找
  5. `-user`：按照文件属主来查找
  6. `-nouser`：查找没有有效属主的文件
  7. `-group`：按照文件的所属组来查找
  8. `-nogroup`：查找没有有效的所属组文件
  9. `-type`：按照文件类型来查找（`f`是文件类型，`d`是目录类型，`l`是链接类型）
  11. `-amin`：在过去指定分钟被读取过的文件（`+` 表示指定分钟以前，` -`表示指定分钟以内）
  12. `-atime`：在过去指定天数被读取过的文件（`+` 表示指定天数以前，` -`表示指定天数以内）
  13. `-cmin`：在过去指定分钟（创建、修改、变更属性）的文件（`+` 表示指定分钟以前，` -`表示指定分钟以内）
  14. `-ctime`：在过去指定天数（创建、修改、变更属性）的文件（`+` 表示指定天数以前，` -`表示指定天数以内）
  15. `-min `：在过去指定分钟修改过内容的文件（`+` 表示指定分钟以前，` -`表示指定分钟以内）
  16. `-mtime `：在过去指定天数修改过内容的文件（`+` 表示指定天数以前，` -`表示指定天数以内）
  17. `-size`：按照文件大小来查找（单位 `c k M G`，`+` 表示大于，` -`表示小于）
  18. `-maxdepth` ：指定搜索时向下递归的层数
- 输出方式：
  1. `-print`：默认，输出查询结果到标准输出( `-print0`表示每条结果后面不使用换行使用`\0`分隔)
  2. `-ecex`：对查询结果进行执行操作， 指定的命令中使用`{}`表示当前查询到的一条文件，`+`表示等到查询到所有文件再执行
  3. `-ok`：同上，删除文件时会有提示

```shell
# 查找当前文件小写字母开头的.txt文件
find -name '[a-z]*.txt'

# 按权限查找指定目录的文件，'/a=x'表示所有用户可执行的文件，'/'表示只匹配权限的某个子集
find -perm u=rwx,g=rx,o=rx 
find -perm 644
find /usr -perm a=rx
find -type f -perm /a=x

# 将home目录下所有的 MP3 音频文件复制到指定移动存储设备
find ~ -type f -name '*.mp3' -exec cp {} /media/MyDrive ';'

# 将home目录下所有的 MP3 音频文件添加到指定压缩包中
find ~ -type f -name '*.mp3' -exec tar -czvf music.tar.gz {} +
```

### echo：输出所接受的内容

- 基本语法：`echo [选项] 文本`
- 选项：
  1. `-n`：不要输出命令结束时自动的换行
  2. `-e`：启用反斜线转义解释
- 字体设置
  1. `\033[颜色1;颜色2m`:  `\033[`表示转译序列开始，`m`表示转译序列结束，后面的文字使用指定的颜色配置
  2. `\033[颜色1m`: 同上，只定义前景色或背景色（多个效果用`;`隔开）
  3. `\E[颜色1m` `\e[颜色1m`：同上效果
- 字体转译符
  1. `[0m`: 正常
  2. `[1m`: 粗体
  3. `[4m`: 字体加上下划线
  4. `[7m`: 逆转前景和背景色
  5. `[8m`: 不可见字符
  6. `[9m`: 跨行字体
  7. `[30m`: 灰色字体
  8. `[31m`: 红色字体
  9. `[32m`: 绿色字体
  10. `[33m`: 棕色字体
  11. `[34m`: 蓝色字体
  12. `[35m`: 紫色字体
  13. `[36m`: 浅蓝色字体
  14. `[37m`: 浅灰字体
  15. `[38m`: 黑色字体
  16. `[40m`: 黑色背景
  17. `[41m`: 红色背景
  18. `[42m`: 绿色背景
  19. `[43m`: 棕色背景
  20. `[44m`: 蓝色背景
  21. `[45m`: 紫色背景
  22. `[46m`: 浅蓝色背景
  23. `[47m`: 浅灰色背景

```shell
# 支持通配符输出当前目录的文件
echo The TXT files are: *.txt

# 显示命令输出,使用$(command)表达式将命令输出作为参数传递给echo
echo "The date is: $(date +%D)"

# 输出红色的'Magic of Linux',后续输出恢复正常
echo -e "\033[31mMagic of Linux\033[0m end"

# 在脚本中定义好颜色变量，方便使用
RED='\E[1;31m'
GREEN='\E[1;32m'
YELOW='\E[1;33m'
BLUE='\E[1;34m'
PINK='\E[1;35m'
RES='\E[0m'
# 实际使用
echo -e "${RED}Red color${RES}"
```

### sed：文件流编辑器，一次处理一行，把当前行的内容存储在临时缓冲区（模式空间）进行处理，处理完成后把缓冲区的内容输出到标准输出，接着处理下一行，直到文件末尾，原文件默认不会改变。

- 基本语法：`sed [选项] 文件名`
- 选项：
  1. `-e`：指定的script来处理输入的文本文件（可以省略，但是后面的脚本需要用引号包含）
  2. `-f`：指定的script文件来处理输入的文本文件
  3. `-n`：仅显示处理后的结果
  4. `-i`：是当前修改作用于原文件
- 脚本动作：
  1. `a\`： 在当前行后面加入一行指定文本
  2. `i\` ：在当前行上面插入一行指定文本
  3. `d` ：从模版块删除行
  4. `c\` ：用新的文本改变指定行的文本
  5. `s/正则表达式/新内容/g`：用新内容替换正则表达式匹配的内容。（‘/’分隔符可以替换为其他符号；‘g’为全局替换，用于当前行多个匹配内容时全部替换）
  6. `h`：拷贝模板块的内容到内存中的缓冲区
  7. `p`：指定要打印的行
  8. `q`：退出脚本

```shell
# 测试数据 test.txt
dong shen
guan zhen
wo  wo
lai  lai
le  le1

# 将'abc'插入到第2行的下一行
sed '2a\abc' test.txt

# 将'abc'插入到第2行的下一行,将'def'跟着换行插入
sed '2a\abc\
def' test.txt

# 删除第2和第3行
sed '2,3d' test.txt

# 删除第3行到最后一行
sed '3,$d' test.txt

# 替换最后一行为'good' 
sed '$c\good' test.txt

# 搜寻字母e行并显示
sed -n '/e/p' test.txt

# 搜寻字母e的行并删除
sed '/e/d' test.txt

# 搜寻'en'并替换为'eng'
sed 's/en/eng/' test.txt

# 搜寻'o'并替换为'oo',
sed 's|o|oo|g' test.txt

# 搜寻字母u所在行并执行替换脚本，输出结果
sed -n '/u/{s/en/eng/;p;q}' test.txt

# 连续命令
sed -e '$a\good' -e 's/o/oo/g' test.txt
```
